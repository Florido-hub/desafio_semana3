# Product

This project is an automated API test suite for the **ServeRest** REST API — a public e-commerce backend used for QA training purposes.

The API simulates a basic online store with four domains:
- **Login** — authentication and token retrieval
- **Usuários** (Users) — user registration and management
- **Produtos** (Products) — product catalog management (admin-only writes)
- **Carrinhos** (Carts) — shopping cart operations

The base URL under test is: `https://compassuol.serverest.dev/`

Tests are written in Portuguese (resource names, variable names, comments) reflecting the Brazilian QA training context this project belongs to.
